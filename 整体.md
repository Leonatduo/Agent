Agent开发是智能体开发, 简单理解就是智能体像人一样思考决策， 而不是像传统开发一样只进行工具调用， 通过LLM识别自然语义， 解析， 自主决定然后调用工具完成相应功能的整个过程。

整体流程： 
开发者写好对应的Tools(描述， 入参和出参格式定义清晰)， 然后接通大模型接口， 用户输入提示词之后大模型解析提示词识别出要调用哪个tool， 然后根据返回的结果和分析判断是否能够解决问题，
如果可以解决问题的话就直接输出结果，否则带着上下文和结果继续分析。

reACT
Thought 大模型根据用户输入进行分析和思考， 看现有信息是否足够回答用户问题， 如果可以的话就直接返回结果， 不行的话就根用户问题和Tool的description调用工具
Action 大模型调用Tools, 然后根据对应的入参和出参约束生成相应的结果
Observation 看结果是否能回答用户问题， 如果不能的话带着结果+ 用户提问 + 多轮的（Thought + Action + Observation）进行下一轮的评估
